- IMG: Cosine and sine diagrams
- $$\cos_n = \frac{1}{2}(C_n + C_{-n})$$
    - Adding two circular motions in opposite directions
    - Adds constructively in the horizontal direction, cancels out vertically, hence the result is real
    - Starts aligned, so max value at 0
    - cancels out at $\pi/2$
- $$\sin_n = \frac{1}{2i}(C_n - C_{-n})$$
    - Subtracting two circular motions in opposite directions
    - Result is vertical, so the $1/i = -i$ rotates it a quarter turn clockwise so it points along the real line
    - Difference is 0 at the angle 0
    - Difference is at a maximum at $\pi/2$
